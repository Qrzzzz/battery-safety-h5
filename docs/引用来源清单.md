# 引用来源清单

正式提交前建议至少补齐 3–5 个权威来源。可优先使用以下类型：

1. 国家或地方消防救援部门发布的锂电池、电动车充电安全科普材料。
2. 市场监管、应急管理、消防、工信等部门发布的电池产品安全提示。
3. 高校教材或公开课程中关于锂离子电池工作原理、热失控机理的内容。
4. 期刊论文或综述中关于锂离子电池热失控、隔膜失效、电解液分解的基本机理表述。
5. 产品安全说明书中关于充电、储存、鼓包、进水、过热处理的规范建议。

## 引用格式建议

- 机构/作者：《标题》，发布平台，发布日期，访问日期。
- 网页来源需保留完整 URL。
- 图片、图标、音乐、字体、视频模板等素材必须标注来源和授权方式。

## 当前原型中的科学内容待核验点

- “热失控”定义与触发条件。
- 鼓包、异味、异常发热的风险提示。
- 过充、短路、挤压、高温等风险因素。
- 异常电池的处置原则。
- 不同温度区间的表达是否过度具体。正式提交前建议避免写死未经来源支持的精确阈值。
