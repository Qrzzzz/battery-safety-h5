# 4分钟以内演示视频脚本

## 0:00–0:15 片头

画面：作品首页。
旁白：大家好，我们的作品是《别让电池失控：锂电池热失控安全科普互动页》。这是一款面向大学生和公众的 H5 科普作品，目的是帮助用户识别锂电池异常风险，理解热失控的基本原理。

## 0:15–0:55 问题导入

画面：首页滚动到核心问题。
旁白：锂电池已经广泛用于手机、充电宝、电动车和各类便携设备。很多人知道电池会发热，但分不清普通发热和危险信号。我们的作品从日常场景出发，把专业安全知识转化为互动判断。

## 0:55–1:35 风险判断模块

画面：点击三个场景卡片。
旁白：第一个模块是风险判断。用户可以选择不同场景，系统会给出风险等级和解释。例如，轻微发热通常不等于事故；但如果充电宝已经鼓包还继续使用，就属于明显的高风险行为。

## 1:35–2:15 温度滑杆模块

画面：拖动温度滑杆。
旁白：第二个模块用温度滑杆表现风险递进。随着温度升高，页面会提示内部副反应、隔膜受损、短路和连锁升温等风险，让用户直观看到热失控不是突然发生的，而是多种因素叠加后的结果。

## 2:15–2:55 应急处理模块

画面：选择错误和正确选项。
旁白：第三个模块是应急选择。作品强调发现鼓包、异味、异常发烫时，应立即停止使用，远离可燃物，并联系专业人员处理，不能继续充电、私自拆解或用不规范方式降温。

## 2:55–3:35 总结与技术说明

画面：点击生成学习总结，展示代码目录。
旁白：最后，系统会生成学习总结，方便用户记忆和传播。技术上，本作品采用 HTML、CSS 和 JavaScript 实现，是纯静态页面，可在线访问，也可以离线打包提交，兼容主流浏览器和移动设备。

## 3:35–4:00 片尾

画面：作品标题页或总结页。
旁白：我们希望用轻量互动的形式，让更多人理解锂电池安全知识，减少日常使用中的风险误区。谢谢观看。
