# 设计系统说明

## 视觉方向

本项目由荧光绿色赛博风重构为 **Industrial Safety Diagnostic / 工业安全诊断风**。整体目标不是“更炫”，而是让页面更像公共安全教育产品、实验室诊断界面与工业安全监测系统。

## 色彩系统

- `--bg #05070A`：石墨黑背景，降低娱乐化观感。
- `--surface #0B1117` / `--surface-2 #111A22` / `--surface-3 #17232D`：卡片、面板和层级背景。
- `--text #E8ECEA`：主文本。
- `--text-soft #A7B0AD` / `--text-muted #6F7A78`：说明文字与弱信息。
- `--accent #4FC3C7`：诊断青，用于主按钮、状态线和当前进度。
- `--warning #D8A742`：工业琥珀，用于注意、异常、误判提示。
- `--danger #E15A46`：氧化红，用于 critical / emergency 状态。
- `--safe #7FB069`：安全选择和正确结果。

旧版荧光绿不再作为主视觉色，避免“游戏化”和廉价发光效果。

## 字体与排版

- 正文使用系统中文字体栈，保障 GitHub Pages 与离线环境稳定。
- 状态码、页码、风险等级、进度点使用等宽字体。
- 标题保留大字号，但降低负字距和描边效果，使中文标题更稳重。

## 组件

- `.diagnostic-panel`：诊断面板基类，用于封面监测、设备诊断和趋势仪表。
- `.status-chip`：小型状态标签。
- `.risk-meter`：风险计数和状态文本。
- `.protocol-card`：机理链路和应急协议卡。
- `.source-card`：证据页来源卡片。
- `.thermal-map`：热成像区域提示。
- `.mono`：等宽字体工具类。

## 页面设计逻辑

1. **封面页**：右侧由电池图标升级为电芯状态监测面板，展示 STATUS、RISK LEVEL、SESSION。
2. **热失控解释页**：使用 Trigger → Heat Accumulation → Chain Reaction → Vent / Smoke / Fire 风险传导图。
3. **风险信号页**：多选题变为诊断台，左侧设备轮廓、扫描与风险状态，右侧信号卡片。
4. **场景判断页**：A/B/C 题升级为 P1/P2/P3 风险分诊卡，强调优先级。
5. **升温趋势页**：滑杆变为阶段仪表，热区随阶段从冷色到琥珀、橙红、红色递进。
6. **安全处置页**：用应急决策模拟器输出判断、风险原因与推荐动作；STOP / DISTANCE / REPORT 作为 SOP。
7. **完成页**：结业卡弱化为正式安全提示摘要卡，PNG 使用深色背景、细线边框和诊断青强调。
8. **来源页**：由长链接列表升级为证据卡，展示类型、标题、机构 / 期刊、年份、用途说明和查看来源按钮。

## 动效原则

动效只用于表达状态变化：扫描线、页面切换、危险热区递进。项目保留 `prefers-reduced-motion`，减少强闪烁、强抖动和霓虹脉冲。
