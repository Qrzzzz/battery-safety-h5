# 《失控之前》锂电池热失控安全科普 H5

面向大学生和普通公众的中文互动科普作品。作品通过生活场景、风险信号识别、场景判断、升温趋势演示和安全处置选择，帮助用户理解锂电池异常风险，并掌握“看信号、不试探、先避险”的基本原则。

## 在线访问

https://qrzzzz.github.io/battery-safety-h5/

## 作品定位

这是一个纯静态 H5 科普作品，适合课堂展示、手机端访问和比赛提交。项目不依赖后端服务，不需要安装复杂环境。

## 主要功能

- 风险信号多选判断
- 生活场景风险判断
- 升温趋势滑杆演示
- 安全处置选择反馈
- 安全提示卡 PNG 生成
- 资料依据展示

## 页面结构

1. 封面
2. 热失控是什么
3. 识别风险信号
4. 场景判断
5. 升温趋势
6. 安全处置
7. 安全提示卡
8. 资料依据

## 技术栈

- HTML
- CSS
- JavaScript
- Canvas PNG 生成
- GitHub Pages 静态部署

## 本地运行

直接双击 `index.html` 即可运行。

也可以在项目根目录执行：

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:4173
```

## 内容依据

作品参考政府文件、消防科普、厂商安全说明和锂离子电池热失控相关综述论文。具体来源见页面“资料依据”部分。
