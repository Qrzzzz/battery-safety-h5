# 《失控之前》锂电池热失控安全科普 H5

《失控之前》是一套面向大学生与普通公众的纯静态公共安全交互指南，定位为“锂电池安全监测平台 + 消防安全科普 + 实验室诊断界面”的轻量化 H5。作品通过分页学习、风险信号诊断、场景分诊、升温趋势仪表和应急决策模拟，帮助用户建立“看信号、不试探、先避险”的安全认知。

## 在线访问

https://qrzzzz.github.io/battery-safety-h5/

## 作品定位

- **公共安全教育产品**：避免娱乐化表达，强调异常识别、风险优先级和规范处置。
- **工业安全诊断风格**：使用石墨黑、诊断青、工业琥珀和警示红构建克制、可信的界面语言。
- **低门槛静态部署**：不依赖 React、Vue、Vite、Tailwind、CDN 或外部字体，适合 GitHub Pages 直接托管。

## 主要功能

1. 分页式安全学习与顶部 / 底部进度导航。
2. 风险信号多选诊断，识别鼓包、异味、异响、持续发烫、烟气等异常。
3. 场景风险分诊，区分 P1 立即停用、P2 环境风险、P3 观察项。
4. 升温趋势滑杆，演示热失控风险阶段变化。
5. 应急决策模拟，输出选项判断、风险原因和推荐动作。
6. STOP / DISTANCE / REPORT 三步 SOP 安全处置卡。
7. 完成页生成 1200 × 1600 PNG 安全提示卡。
8. 资料来源证据页，保留政府文件、消防科普、产品说明和综述论文链接。

## 页面结构

| 页码 | 页面 | 交互目的 |
| --- | --- | --- |
| 0 | 封面 / 电芯状态监测面板 | 建立诊断产品氛围与学习目标 |
| 1 | 热失控解释 | 理解 Trigger → Heat Accumulation → Chain Reaction → Vent / Smoke / Fire |
| 2 | 风险信号识别 | 多选识别异常信号并反馈误判 |
| 3 | 场景判断 | 理解风险优先级与电池本体异常 |
| 4 | 升温趋势 | 通过滑杆观察阶段化风险累积 |
| 5 | 安全处置 | 选择正确应急动作并学习 SOP |
| 6 | 完成页 | 生成安全提示 PNG 摘要卡 |
| 7 | 来源页 | 查阅证据来源与用途说明 |

## 技术栈

- HTML5 语义化结构
- CSS3 自定义属性、响应式布局、`prefers-reduced-motion`
- 原生 JavaScript
- Canvas PNG 生成
- 无构建流程、无第三方依赖、无 CDN

## 本地运行方式

直接双击 `index.html` 即可运行。

如需使用本地 HTTP 服务测试，可在项目根目录执行：

```bash
python -m http.server 4173 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:4173/
```
