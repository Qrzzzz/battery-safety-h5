# 《失控之前》锂电池热失控安全科普 H5

面向大学生和普通公众的静态互动科普作品，建议体验时长 3–5 分钟。作品采用八页式学习流程，通过风险信号识别、场景判断、升温趋势演示和应急决策，帮助用户理解锂电池热失控的基本原理及安全处置原则。

## 本地运行

最简单的方式：直接双击 `offline-site/index.html`。

如需使用本地 HTTP 服务测试，可在 `offline-site` 目录执行：

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

然后访问 `http://127.0.0.1:4173/`。

## 文件结构

- `offline-site/index.html`：页面结构与科普内容
- `offline-site/assets/style.css`：视觉系统、响应式布局和动画
- `offline-site/assets/app.js`：逐页导航、互动评分和 PNG 结业卡
- `docs/`：申报、视频脚本、引用来源和提交材料文档

页面不依赖外部框架、字体、图片或后端，可离线运行。最后一页列出了政府文件、消防科普、产品说明和综述论文来源。正式参赛前应根据比赛要求补充团队信息、作品编号、效果截图和演示视频。
